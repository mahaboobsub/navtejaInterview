# navtejaInterview

Java Fullstack Interview Question Bank.

## Features
- 9 modules (Core Java, SQL, Adv Java, Hibernate, Spring Core, Backend, DSA, HTML/CSS, JavaScript)
- 400+ interview questions with detailed, well-formatted answers
- Premium Dark Mode design with glassmorphism aesthetics
- Instant search and filtering across all questions
- Copy questions instantly with one click
- Completely offline and responsive layout
